# Shared UI Components Library

欢迎使用我们的 UI 组件库！这是一个基于 React + Tailwind CSS + shadcn/ui 的组件库，旨在提供一套美观、易用且可定制的 UI 组件。

## 组件列表

目前，我们提供以下组件：

- [Button](/docs/ui-button--docs) - 多样式按钮组件
- [Header](/docs/ui-header--docs) - 响应式页面头部组件
- [Checkbox](/docs/ui-checkbox--docs) - 复选框组件
- [Avatar](/docs/ui-avatar--docs) - 头像组件
- [Breadcrumb](/docs/ui-breadcrumb--docs) - 面包屑导航组件
- [Label](/docs/ui-label--docs) - 表单标签组件
- [Select](/docs/ui-select--docs) - 下拉选择组件
- [Skeleton](/docs/ui-skeleton--docs) - 骨架屏加载组件
- [Switch](/docs/ui-switch--docs) - 开关组件
- [Textarea](/docs/ui-textarea--docs) - 文本域组件
- [Toaster](/docs/ui-toaster--docs) - 消息提示组件

## 设计原则

我们的组件库遵循以下设计原则：

- **简洁性**：组件 API 简单明了，易于使用
- **可定制性**：支持主题定制和样式覆盖
- **可访问性**：符合 WCAG 标准，支持键盘导航和屏幕阅读器
- **响应式**：适应不同屏幕尺寸和设备

## 开始使用

### 安装

```bash
pnpm add @shared-libs/ui
```

### 使用示例

```jsx
// 按钮组件示例
import { Button } from '@shared-libs/ui';

function ButtonExample() {
  return <Button variant="primary">点击我</Button>;
}

// 头部组件示例
import { Header } from '@shared-libs/ui';

function HeaderExample() {
  return (
    <Header
      leftContent={<div className="font-bold">网站名称</div>}
      menu={
        <nav className="flex space-x-4">
          <a href="#">首页</a>
          <a href="#">关于</a>
        </nav>
      }
      rightContent={<Button size="sm">登录</Button>}
    />
  );
}

// 复选框组件示例
import { Checkbox } from '@shared-libs/ui';

function CheckboxExample() {
  return (
    <div className="flex items-center space-x-2">
      <Checkbox id="terms" />
      <label htmlFor="terms" className="text-sm font-medium">
        接受条款和条件
      </label>
    </div>
  );
}

// 头像组件示例
import { Avatar, AvatarImage, AvatarFallback } from '@shared-libs/ui';

function AvatarExample() {
  return (
    <Avatar>
      <AvatarImage src="https://github.com/shadcn.png" alt="@shadcn" />
      <AvatarFallback>CN</AvatarFallback>
    </Avatar>
  );
}

// 开关组件示例
import { Switch, Label } from '@shared-libs/ui';

function SwitchExample() {
  return (
    <div className="flex items-center space-x-2">
      <Switch id="airplane-mode" />
      <Label htmlFor="airplane-mode">飞行模式</Label>
    </div>
  );
}
```
